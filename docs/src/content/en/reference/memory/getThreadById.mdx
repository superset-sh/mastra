# Memory.getThreadById()

The `.getThreadById()` method retrieves a specific thread by its ID.

## Usage Example

```typescript
await memory?.getThreadById({ threadId: 'thread-123' })
```

## Parameters

<PropertiesTable
  content={[
{
name: "threadId",

isOptional: false,
},
]}
/>
