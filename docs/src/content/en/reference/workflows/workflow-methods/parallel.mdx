# Workflow\.parallel()

The `.parallel()` method executes multiple steps in parallel.

## Usage example

```typescript
workflow.parallel([step1, step2])
```

With `allSettled` mode to continue even when individual steps fail:

```typescript
workflow.parallel([step1, step2], { mode: 'allSettled' })
```
