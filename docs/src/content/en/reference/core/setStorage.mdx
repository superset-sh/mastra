# Mastra.setStorage()

The `.setStorage()` method is used to set the storage instance for the Mastra instance. This method accepts a single `MastraCompositeStore` parameter.

## Usage example

```typescript
mastra.setStorage(
  new LibSQLStore({
    id: 'mastra-storage',
    url: ':memory:',
  }),
)
```
